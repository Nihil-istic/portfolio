# Popsicles


You have a box of popsicles and you want to give them all away to a group of
brothers and sisters. If you have enough left in the box to give them each an
even amout you should go for it! If not, they will fight over them, and you
should eat them for yourself later.


## Task


Given the number of siblings that you are giving posicles to, determine if you
can give them each one an even amount or if you shouldn't mention the posicles
at all.


## Input Format


Two integer values, the first one represents the number of siblings, and the
second one represents the number of popsicles that you have left in the box.


## Output Format


A string that says 'give away' if you are giving them away, or 'eat them
yourself' if you will be eating them yourself.


## Sample Input
3

9


## Sample Output
give away
